3   Cálculo de dominios

Para calcular el dominio de una función f dada por una fórmula “eliminaremos” aquellos valores de x que hagan imposible realizar alguna operación matemática. Esto ocurre para:

  • Denominadores que se anulan
  • Raíces de índice par de números negativos
  • Logaritmos de 0 o de números negativos

Los cálculos más usuales de dominios se refieren a las funciones siguientes:

  • Funciones polinómicas
    El dominio de toda función polinómica es ℝ, ya que al sustituir un número real cualquiera x ∈ ℝ, siempre va a existir f(x).
  • Funciones racionales
    Para una función de la forma f(x) = P(x) Q(x),

    donde tanto P(x) como Q(x) son polinomios, su dominio es el conjunto de valores que no anulan el denominador, es decir

    Dom(f) = {x ∈ ℝ|Q(x)≠0}.

    Cuando simplifiques una función racional, el dominio debe coincidir con el de la función original, y no debes caer en la tentación de recalcularlo.

  • Funciones irracionales
    Para una función irracional del tipo f(x) = g(x)n,

    su dominio depende de si el índice n es par o impar.

    • Si n es impar
      En estos casos, la raíz no impone ninguna restricción adicional al dominio, por lo que coincidirá con el del radicando g(x).
    • Si n es par
      En estos casos la raíz impone que los valores del radicando siempre sean mayores o iguales que cero, con lo que Dom(f) = {x ∈ Dom(g)|g(x) ≥ 0}.

  • Funciones exponenciales
    En una función de la forma f(x) = eg(x),

    la exponencial no impone ninguna restricción adicional al dominio, con lo que el dominio de f coincidirá con el dominio de la función g en el exponente.

  • Funciones logarítmicas
    Para una función de la forma f(x) = log(g(x)),

    el logaritmo impone que el argumento debe de ser un número positivo, por tanto

    Dom(f) = {x ∈ Dom(g)|g(x) > 0}.

  • Funciones trigonométricas
    El dominio de las funciones del tipo f(x) = sin ⁡ (g(x)) y f(x) = cos ⁡ (g(x)), es el dominio de g(x). Las funciones del tipo f(x) = tan ⁡ (g(x)) tienen por dominio: Dom(f) = {x ∈ ℝ|g(x)≠π 2 + kπ, k ∈ ℤ}

Problema 3

Calcula el dominio de las siguientes funciones:

  • f(x) = 2x 3x − 2
  • f(x) = 6x x2 − 16
  • f(x) = x2 − 1
  • f(x) = 3 −x + 2
  • f(x) = x + 5 x − 7
  • f(x) = x2 + x
  • f(x) = log ⁡ (−2x2 − 10x − 8)
  • f(x) = ex
  • f(x) = e−5x
Solución:
  • ℝ −{2 3}
  • ℝ −{−4, 4}
  • (−∞,−1] ⋃ ⁡ [1, +∞])
  • (−∞, 2)
  • (−∞,−5] ⋃ ⁡ [7, +∞])
  • (−∞,−1] ⋃ ⁡ [0, +∞])
  • (−4,−1)
  • ℝ
  • ℝ
