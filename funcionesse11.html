11   Función inversa

Definición 18. Dada una función f, se denomina función inversa de f, si existe, y se simboliza con f−1, la función que cumple:

f−1(y) = x↔f(x) = y

La función inversa de f−1 es, a su vez, f : (f−1)−1 = f. Por eso decimos, simplemente, que las funciones f y f−1 son inversas.

Una función y su inversa verifican las siguientes propiedades:

  • La composición de una función f con su inversa f−1 es la función identidad, I(x) = x: (f−1 ∘ f)(x) = f−1(f(x)) = I(x)

    (f ∘ f−1)(x) = f(f−1(x)) = I(x)

  • Las gráficas de f y de f−1, referidas al mismo sistema de coordenadas, son simétricas respecto de la bisectriz del primer cuadrante.

Para calcular la función inversa de una función dada f debemos de seguir el siguiente procedimiento:

  1. Se escribe la función con x e y.
  2. Se despeja la variable x en función de la variable y.
  3. Se intercambian las variables.

Problema 30

Halla la función inversa de f(x) = 2x + 3.

Solución:

  1. Intercambiamos las variables x e y en la expresión y = 2x + 3, con lo que resulta x = 2y + 3.
  2. Despejamos x, con lo que en este caso se obtiene x = 2y + 3.
  3. Despejamos y, con lo que obtenemos y = x − 3 2 . Luego f−1(x) = x − 3 2 .

Comprobamos que la composición de las dos funciones hace corresponder a cada x el mismo x:

(f−1 ∘ f)(x) = f−1(f(x)) = f−1(2x + 3) = 2x + 3 − 3 2 = x

(f ∘ f−1)(x) = f(f−1(x)) = f(x − 3 2 ) = 2 ⋅ x − 3 2 + 3 = x

Problema 31

En cada caso, calcula la función inversa de la dada:

  • f(x) = x2 − 1 2
  • h(x) = log ⁡ (x)
  • j(x) = x2 + 5
