1   Concepto de función real

Definición 1. Dados dos conjuntos de números reales A y B, una función de A en B, es una correspondencia por la cual a cada número real x de A le corresponde un único número real y = f(x) de B.

Se puede representar por:

f : A→B x↦y = f(x)

  • x: número real, perteneciente a A, que recibe el nombre de variable independiente.
  • y = f(x): número real perteneciente a B, recibe el nombre de variable dependiente. Se dice que y = f(x) es la imagen de x por la función f.
  • Como tanto x como y = f(x) son números reales, se dice que f es una función real de variable real.
  • Al conjunto A de valores que puede tomar la variable independiente x se llama dominio de f y se denota Dom(f).
  • Al subconjunto de B formado por los valores que puede tomar la variable dependiente y = f(x) se le llama imagen o recorrido de f y se denota Im(f).
  • La gráfica de f está formada por las parejas de valores (x,y) siendo x ∈ Dom(f) e y = f(x).

Cuando una función f viene definida mediante una fórmula y = f(x), si no se indica otra cosa, se sobreentiende que su dominio Dom(f) es el conjunto de valores de x para los que la fórmula tiene sentido y puede calcularse f(x).
