4.3   Ejercicios

Problema 5

Estudia la simetría de las siguientes funciones indicando las operaciones que has realizado. Para ayudarte, puedes servirte de la representación gráfica de las mismas:

  • f(x) = 3x − x3
  • f(x) = x4 − 2x2 − 8
  • f(x) = x6 + x4 − x2
  • f(x) = x5 + x3 − x
  • f(x) = x ⋅|x|
  • f(x) = |x|− 1
  • f(x) = x2 1 − x2
  • f(x) = x 1 − x2
  • f(x) = x4 + 1 x2
  • f(x) = x2 2 − x

Solución:
  • f(−x) = 3(−x) − (−x3) = −(3x − x3) = −f(x) → Función simetría impar
  • f(−x) = (−x)4 − 2(−x)2 − 8 = f(x) → Función simetría par
  • f(−x) = (−x)6 + (−x)4 − (−x)2 = x6 + x4 − x2 =→ Función simetría par
  • f(−x) = (−x)5 + (−x3) − (−x) = −x5 − x3 + x = −f(x) → Función simetría impar
  • f(−x) = −x|x| = −x|x| = −f(x) → Función simetría impar
  • f(−x) = |− x|− 1 = |x|− 1 = f(x) → Función simetría par
  • f(−x)0 (−x)2 1 − (−x)2 = x2 1 − x2 = f(x) → Función simetría par
  • f(−x) = (−x) 1 − (−x)2 = −x 1 − x2 = −f(x) → Función simetría impar
  • f(−x) = (−x)4 + 1 (−x)2 = f(x) → Función simetría par
  • f(−x) = (−x)2 2 − (−x) = x2 2 + x → No presenta simetría

Problema 6

¿Qué tipo de simetría presentan las siguientes funciones?:

  • f(x) = x − 1
  • g(x) = 1 x
  • h(x) = x2 − 4 x2 + 1

Solución:
  • f(x) → no simétrica
  • g(x) → simetría impar
  • h(x) → simetría par
