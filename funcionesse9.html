9   Operaciones con funciones

Sean dos funciones f y g definidas en el mismo subconjunto D de los números reales, se definen las siguientes operaciones entre ellas:

  • Adición: la suma de f y g es la función f + g que, para cualquier x ∈ D, verifica: (f + g)(x) = f(x) + g(x)

  • Sustracción: la diferencia de f y g es la función f − g que, para cualquier x ∈ D, verifica: (f − g)(x) = f(x) − g(x)

  • Multiplicación: el producto de f y g es la función f ⋅ g que, para cualquier x ∈ D, verifica: (f ⋅ g)(x) = f(x) ⋅ g(x)

  • División: el cociente de f y g es la función f g que, para cualquier x ∈ D, con g(x)≠0, verifica: (f g )(x) = f(x) g(x)

Las operaciones con funciones verifican las mismas propiedades que las operaciones con números reales, ya que se definen utilizando las operaciones con sus imágenes, que son números reales.

El dominio de las funciones f + g, f − g, f ⋅ g y f g es la intersección de los dominios de f y g, con la salvedad de que en el caso f g, los valores de x que anulan el denominador no pertenecen al dominio.

Problema 24

Dadas las funciones f(x) = 5x + 6 y g(x) = 3x2 − 4x + 8, calcula la suma f + g y la resta f − g.

Solución:
  • (f + g)(x) = f(x) + g(x) = (5x + 6) + (3x2 − 4x + 8) = 3x2 + x + 14
  • (f − g)(x) = f(x) − g(x) = (5x + 6) − (3x2 − 4x + 8) = −3x2 + 9x − 2

Problema 25

Dadas las funciones f(x) = 12x3 + 15x2 − 6x y g(x) = 3x, calcula el producto f ⋅ g y la división f g.

Solución:
  • (f ⋅ g)(x) = f(x) ⋅ g(x) = (12x3 + 15x2 − 6x) ⋅ (3x) = 36x4 + 45x3 − 18x2
  • (f g )(x) = f(x) g(x) = (12x3 + 15x2 − 6x) (3x) = 4x2 + 5x − 2

Problema 26

Dadas las funciones f(x) = x, g(x) = x − 4 y h(x) = 2 3 − x, calcula

  • f + g
  • g + h
  • f g
  • g f
  • f ⋅ g
  • g h
Solución:
  • (f + g)(x) = f(x) + g(x) = x + (x − 4) = x + x − 4
  • (g + h)(x) = g(x) + h(x) = (x − 4) + 2 3 − x = x − 4 + 2 3 − x
  • (f g )(x) = f(x) g(x) = x x − 4
  • (g f )(x) = g(x) f(x) = x − 4 x
  • (f ⋅ g)(x) = f(x) ⋅ g(x) = x ⋅ 2 3 − x = 2x 3 − x
  • (g h)(x) = g(x) h(x) = x − 4 2 3 − x = (x − 4)(3 − x) 2
